Before running any Google Cloud scripts, do the following:
    1. Add the json file containing all google authentication credentials here.
    2. Run the following line to set an environment variable
        - export GOOGLE_APPLICATION_CREDENTIALS="enter full path to .json file here"
        - Example: export GOOGLE_APPLICATION_CREDENTIALS="/GoogleVisionAPI/GoogleCloudCredentials/myapicreds.json"
    3. You need to re-run this every session
